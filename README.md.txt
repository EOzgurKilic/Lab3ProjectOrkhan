## How to Run

1. **Install Live Server extension** (in VS Code):
- Go to Extensions sidebar (`Ctrl+Shift+X`)
- Search for **Live Server**
- Click **Install**

2. **Open the project folder** in VS Code.

3. **Right-click on `index.html`** and choose:
> **"Open with Live Server"**

5. You’ll see the homepage in your browser.
- Click a video to view it on a separate page
- Click **Download This Clip** to download it locally

---

## Features

- Fully responsive layout (grid-based)
- Unique dark-themed styling
- Thumbnail hover effects
- Custom video player pages
- Direct video download using JavaScript

---

## Notes

- Download links use a dynamic `<a>` element created by JavaScript to work properly across browsers.
- Works 100% offline — no external libraries or APIs required.
